# DuckLoader

Thread and Details: https://www.reddit.com/r/DataHoarder/comments/d0ucp6/all_donald_ducks_released_on_finland_will_be_free/

Downloads all the comics from the akuankka.fi page.

### How to use:
#### Yarn (preferred)
```bash
$ yarn
$ node src/downloadComix.js
```

#### NPM

```bash
$ npm install
$ node src/downloadComix.js
```
